# pgh-events-api

API for https://pgh.events

## Getting Started

1.  **Install Netlify CLI.**

    ```shell
    npm install netlify-cli -g
    ```

1.  **Clone this repo.**

    ```shell
    git clone git@github.com:ericsakmar/pgh-events-api.git
    ```

1.  **Install dependencies.**

    ```shell
    cd pgh-events-api
    yarn install
    ```

## Running

```shell
cd pgh-events-api
netlify dev
```
